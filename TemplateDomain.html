You can upload files to server using CuteFTP with instructions:

Here is a comprehensive procedure for uploading your files on our servers.

There are different softwares for uploading and downloading e.g.
CuteFTP, WS_FTP Pro, LeechFTP etc. CuteFTP is the preferred. The
under defined procedure is for CuteFTP.

1. Install Cuteftp (Latest Version 5.0(trial))
2. When you start the program, it will bring a window FTP SITE MANAGER.
3. Now press the button Add Site.
4. Fill the following fields and leave others blank

HOST ADDRESS: {DomainName}
USER NAME: {UserName}
PASSWORD: {Password}

5. From Login Type select "Normal".
6. Select the site just formed and press CONNECT.
7. Once you press connect, you'll be connected to our server with your domain there.
8. In this new window there are 2 parts of window in middle .... on left its your hard disk and on right it will be server.
9. Once you are connected to server you'll see files on server side(RIGHT SIDE) of window;
10. Now browse your hard disk(LEFT SIDE) to the place where the web page you've designed is present. and select all web files.
11. When you select file, go to menubar "TRANSFER", and press upload or press (CTRL+PgUp)shortcut from keyboard. Now the file will start uploading to the server.
12. For downloading, select any file(s), which you want to download,
and go to menubar "TRANSFER", and hit download or press (CTRL+PgDn)shortcut from keyboard. The file will start downloading to your server.
